# 吹矢アラームちゃん（v.1.4.0）

## 📱 使い方
1. 「はじめ」ボタンを押すと、2秒後にカウントダウンがスタート！
2. 2分30秒経過で「30秒前」の案内音声が出ます
3. カウントダウン中に撥ね矢があれば［撥ね矢］ボタンを押してね
3. 3分経過時、「撥ね矢」ボタンが押されているかどうかで分岐：

### ✅ 跳ね矢ボタンを押していない場合（通常）
- 「3分経過」の音声案内に続いて「終了」の音声が出ます
- タイマー・がリセット（03:00に戻る）

### ✅ 跳ね矢ボタンを押した場合（途中対応あり）
- 「3分経過。跳ね矢がありますので、しばらくお待ちください」の音声を再生
- 「終了」の音声やリセットは自動で行わない（アラームは終了）
- 「終了」ボタンをタップしてタイマーをリセットしてね

---

## ホーム画面への追加方法
- 以下のURLをWebブラウザ（safariやchrome）で開く
  https://FukiyaAbeno.github.io/Alarm/
- すると「インストールしますか？」と画面上部に出てくるので、素直にインストールしてね
- スマホのホーム画面（とドロワー）にアプリアイコンが追加されるよ

---

## 💬 注意事項
- スマホがスリープになる／スクリーンオフ（画面真っ暗）になると音声が鳴らずタイマーが止まります
- このアプリを使う際は、スリープにならず画面真っ暗にもならないよう設定してね
- このアプリの画面を表示したままにしてください
- 詳しくは以下のスリープ防止設定方法を参照してくださいませ

---

## 🛠 スリープ防止設定方法

### 📱 iPhone
- 設定 → 画面表示と明るさ → 自動ロック →「なし」

### 📱 Android
- 設定 → ディスプレイ → スリープ（または画面タイムアウト）→「30分」など

---

## 🎯 その他仕様
